{{- /* Demo project-layout override (the crofty theme stays frozen — see site roles). This replaces the theme's home-only language redirect with a PER-PAGE one: it runs on every page, so a shared deep link (e.g. /posts/images-and-figures/) lands a visitor in their language, not just the home. Existence is never guessed. The candidate languages come from THIS page's own .AllTranslations, which Hugo populates only with translations that actually exist — so a visitor is never bounced to a missing page, and an untranslated page simply stays as it is. Order of preference: an explicit `lang` cookie (set by the switcher in baseof.html) wins; otherwise the browser's primary language. Symmetric (en↔ja) and loop-free, because the current page's own language is never a redirect target. Degrades cleanly with JS off — the visitor gets this page's language plus the manual switcher. Emitted in before the stylesheet so the redirect fires before paint. Shown only when a translation exists. */ -}} {{- if gt (len .AllTranslations) 1 }} {{- $cur := .Language.Lang }} {{- $langs := slice }} {{- range .AllTranslations }}{{ if ne .Language.Lang $cur }}{{ $langs = $langs | append (dict "p" .Language.Lang "url" .RelPermalink) }}{{ end }}{{ end }} {{- end }}
