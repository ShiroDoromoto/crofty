Behind this site there is no database and no dashboard. There is a folder. Inside it are Markdown files, one per post, sorted by name.

I can open that folder in any editor, search it with any tool, back it up by copying it somewhere else. It will still make sense in twenty years, because text always has.

A folder of text is humble, and that is the point. It does not need a company to keep running. It just needs me to keep writing.
