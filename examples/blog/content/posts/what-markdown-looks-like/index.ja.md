このページのすべては Markdown で書かれている。それが何をもたらすか、手短に紹介しよう。

## 見出しは構造を明快に保つ

長い文章を、読者が見渡せる節に分けるのに使う。

**太字**、*斜体*、`インラインコード` を、記法を意識せずに書ける。

- リストはただのハイフン
- 1 行に 1 項目
- 入れ子もできる

> 引用は、立ち止まる価値のある一行を際立たせる。

コードは、フェンスで囲む:

```
crofty build
crofty deploy
```

そして [リンク](https://crofty.site) は角括弧の中のただのテキスト。これで道具立ては全部だ
── エディタもプラットフォームも要らず、どのファイルでも読めるテキストだけ。
